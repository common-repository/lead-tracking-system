== Description ==

This is a lead Tracking plugin.Saves Data from Contact form to Database & sends mail to admin. Also Tracks Referral link & saves in database as well as mail.Please read usage.txt on how to use this plugin.

= Plugin's Official Site =

Lead Tracking System ([http://wisdmlabs.com](http://wisdmlabs.com/))


== Installation ==

1. Upload the entire `Lead tracking system` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Contact' menu in your WordPress admin panel.
